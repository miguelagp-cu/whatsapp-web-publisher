# WhatsApp Web Publisher 🌐

Este proyecto utiliza la **Evolution API** para conectar con grupos de WhatsApp, extraer tipos de mensajes específicos y publicarlos automáticamente.

## 🚀 Objetivo
Automatizar la captura y publicación de contenido relevante desde WhatsApp hacia una plataforma web.
